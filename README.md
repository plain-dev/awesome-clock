### 介绍

一个简单的翻页时钟 ⏰，让旧手机重获新生 ♻

### 注意事项 ⚠

- 推荐采用 ***AMOLED*** 屏幕的设备使用，更加省电 🔋
- 如果是 ***AMOLED*** 设备，长时间使用此应用可能会出现**烧屏**现象，因此推荐**旧手机**使用

### 特点

- 纯粹的翻页时钟
- 可对时钟的大小、翻页背景色、文字颜色和小时制等进行设置
- 没有任何特殊权限请求
